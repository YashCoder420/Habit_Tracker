## Method to Install

##### Clone the Repository

`git clone https://github.com/YashCoder420/Habit_Tracker.git`

##### Into the project directory

`cd Habit_tracker`

##### Install NPM dependencies

`npm install`

##### Then start your app

`npm start`

#### The Server should now be running at http://localhost:3000/
